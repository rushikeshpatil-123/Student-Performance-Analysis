# Student-Performance-Analysis
## 📌 Project Overview

This project analyzes student academic performance using Python, Pandas, and Matplotlib. The objective is to evaluate student scores, calculate subject-wise averages, identify top-performing students, validate data quality, and visualize performance trends through charts.

The dataset contains scores of 1000 students in Mathematics, Reading, and Writing along with demographic information.

---

## 🎯 Objectives

* Load and analyze student performance data using Pandas.
* Calculate average marks for Mathematics, Reading, and Writing.
* Identify the top 5 performing students.
* Validate data by checking missing and negative values.
* Visualize student performance using charts.
* Generate meaningful educational insights.

## 🔍 Data Validation

The dataset was validated to ensure:

* No missing values
* No negative marks
* Clean and consistent data for analysis

---

## 📊 Analysis Performed

### Subject-wise Average Scores

The average score was calculated for:

* Mathematics
* Reading
* Writing

### Top Performing Students

Students were ranked based on their overall average score, and the top 5 performers were identified.

### Data Visualization

The following visualizations were created:

* Bar Chart of Subject Average Scores
* Histogram of Student Performance Distribution

---

## 📈 Key Insights

* Reading had the highest average score among all subjects.
* Writing scores closely followed reading performance.
* Mathematics had the lowest average score.
* Several students achieved near-perfect scores across all subjects.
* Most students scored within the mid-to-high performance range.
